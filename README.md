#### `CORS` : 브라우저의 보안 정책 매커니즘. 브라우저에서 HTTP 요청을 현재 도메인과 다른 도메인으로 보냈을 경우, 응답을 받더라도 사용자에게 전달하지 않는 것을 의미.

#### 해결하려면 -> `서버 쪽에서` 특정 도메인으로부터의 요청을 `허락`했음을 브라우저에 알려줘야 함.

---

#### SignUpSection.jsx : `회원가입 폼`, `CORS`에러를 처음 만나는 페이지.
